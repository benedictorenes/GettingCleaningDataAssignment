# CodeBook 

Three files are available in the Tidy Dataset folder

- "Tidy_set.csv": each row is labeled by subject id, activity/activity_id, and kind of set (train/test), and contains a 79 more observations of mean() and std() values from the acceleration, body acceleration an angular velocity vector measurements in each spatial directions, as reported in the Code Book within the original data set. 


- "Reduced_set.txt"" and "Reduced_set.csv": Each row identifies the subject who performed the activity, and the activity. It contains the average of the quantities reported in the previous dataset grouped by activity and subject id.
